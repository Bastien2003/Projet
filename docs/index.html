Présentation Carte Interactive

Vous avez devant vous une carte interactive du monde, centrée sur le sud-ouest de la France. Les points bleus affichés représentent les différentes gares que nous avons analysées (au total 10). En cliquant sur l’un de ces points, un graphique s’affiche et présente le nombre de trains ayant circulé dans cette gare, mois par mois.
